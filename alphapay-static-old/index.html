About AlphaPay

Built by Africans,
for Africa

AlphaPay was built by Team Alpha as part of the Azubi Africa Internship Programme — Project 1: End-to-End Cloud Solution Design & Deployment.

The platform is hosted on Amazon EC2, distributed through an Application Load Balancer, secured with ACM SSL, delivered via Amazon CloudFront CDN, and fully deployed using infrastructure as code with Terraform.
