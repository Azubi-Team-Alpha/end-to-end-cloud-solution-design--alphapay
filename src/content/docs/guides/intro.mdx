AlphaPay's RESTful API gives developers full programmatic control over payments, transfers, wallet management, and cross-border transactions across Africa. This section provides an overview of our API capabilities and how to integrate them into your platform.

## API Capabilities

<Tabs>
  <TabItem label="Transfers">
    Initiate instant money transfers to any bank account or mobile money wallet across 15+ African countries. Support for single transfers, bulk disbursements, and scheduled payments. All transfers settle in under 2 seconds with full webhook notifications.
  </TabItem>
  <TabItem label="Collections">
    Accept incoming payments from customers via our hosted checkout page, inline payment form, or direct API. Supports mobile money, bank cards, and bank transfer. Real-time webhook events on payment success, failure, and refunds.
  </TabItem>
  <TabItem label="Wallets">
    Programmatically create and manage wallets, query balances, retrieve transaction history, and issue virtual cards. Full support for multi-currency wallets in GHS, NGN, KES, UGX, RWF, and USD.
  </TabItem>
  <TabItem label="Cross-Border FX">
    Get real-time mid-market exchange rates across all supported currency corridors, initiate cross-border transfers, and track settlement status via our FX API. Rates are updated every 60 seconds.
  </TabItem>
</Tabs>

## Base URL & Authentication

All API requests are made to:

```
https://api.alphapay.africa/v1/
```

Authenticate by including your API key in the Authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

API keys are available from the **Developer** section of your AlphaPay dashboard. Always use **test-mode keys** during development and **live-mode keys** only in production.

## Supported Mobile Networks

AlphaPay integrates directly with the following networks across our supported countries:

- **Ghana**: MTN Mobile Money, Telecel Cash, AirtelTigo Money
- **Nigeria**: Bank transfers (all commercial banks via NIP)
- **Kenya**: M-Pesa (Safaricom), Airtel Money
- **Uganda**: MTN Mobile Money, Airtel Money
- **Senegal / Côte d'Ivoire**: Orange Money, Wave
- **Rwanda**: MTN Mobile Money, Airtel Money
