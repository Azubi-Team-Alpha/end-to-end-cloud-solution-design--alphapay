About This Project

This web application was built by Team Alpha as part of the Azubi Africa Internship Programme — Project 1: End-to-End Cloud Solution Design & Deployment.

The app is hosted on an Amazon EC2 instance, distributed through an Application Load Balancer, secured with an ACM SSL certificate, and delivered globally via Amazon CloudFront CDN. Static assets are served from Amazon S3.
